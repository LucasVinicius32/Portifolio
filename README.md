### 3º Semestre/2022 • CopiloTTeam
<b>Link do repositório:</b> <a href="https://github.com/CopiloTTeam">Payme  - CopiloTTeam</a>
<p>Parceiro Acadêmico: <a href="https://www.pro4tech.com.br">Pro4tech</a></p> 

- **Problema apresentado:** O cliente, uma empresa de desenvolvimento de software chamada Pro4Tech, precisava de um sistema para gerenciar suas atividades financeiras e comerciais. Eles enfrentavam desafios como o registro manual de títulos e dados de endereço, além da necessidade de criar relatórios manualmente para identificar clientes em dia e em atraso, com base em datas de pagamento, crédito e vencimento.
- **Proposta de Solução:** Foi sugerida uma solução completa que envolve o desenvolvimento de um sistema abrangente. Esse sistema inclui recursos como cadastro de clientes e gerenciamento de títulos com opção de parcelamento. Além disso, integra-se à API dos correios para facilitar a inserção de dados de endereço. A automação do processamento das parcelas oferece maior agilidade e eficiência ao sistema, beneficiando os processos financeiros e comerciais da empresa.
- **Contribuição pessoal:** Nesse API, concentrei meu trabalho principalmente no desenvolvimento do back-end. Devido a um desafio pessoal, por conta do Java para implementar essa parte do sistema. Além disso, contribuí para a organização da equipe, criando tarefas essenciais para o funcionamento adequado do sistema. Assim, pude apoiar tanto na implementação do back-end quanto na gestão e coordenação do projeto.
